# 🎮 Hangman Daily Challenge

A modern web-based Hangman game where players solve **3 new words every day**. The game provides hints, tracks scores, prevents word repetition, and stores progress locally in the browser.

## 📌 Overview

Hangman Daily Challenge is an interactive word-guessing game built using **HTML, CSS, and JavaScript**. Each day, players receive three randomly selected words from different categories such as Technology, Health, School, Travel, Weather, and more.

The game uses a daily challenge system that ensures players get fresh words every day while avoiding recently used words.

---

## ✨ Features

### 🎯 Daily Challenges

* 3 new words every day.
* Automatically changes based on the current date.
* Previously used words are not repeated until the entire word bank is exhausted.

### 💡 Hint System

* Each word comes with a category and a helpful clue.
* Makes gameplay engaging and educational.

### 🎮 Interactive Gameplay

* Virtual keyboard for guessing letters.
* Physical keyboard support.
* Animated hangman drawing.
* Wrong guess animations.

### 📊 Score Tracking

* Points awarded based on remaining attempts.
* Higher accuracy earns higher scores.
* Total daily score displayed.

### 💾 Local Storage Support

* Saves player progress automatically.
* Allows users to continue where they left off.
* Tracks completed daily challenges.

### 📈 Performance Summary

* Displays a summary after all three words are completed.
* Shows:

  * Total score
  * Won/Lost status for each word
  * Daily performance overview

### 📱 Responsive Design

* Works on desktop, tablet, and mobile devices.
* Modern dark-themed user interface.

## 🎲 Game Rules

1. A player receives 3 words each day.
2. Each word has a category and hint.
3. The player has 6 incorrect attempts.
4. Correct guesses reveal letters.
5. The game ends when:

   * All letters are guessed (Win)
   * 6 wrong attempts are used (Lose)
6. Points are awarded for successful guesses.
